# Case-Social-Connect
Repositório que servirá como 1º Checkpoint da disciplina de WebDev da FIAP
TURMA 1ESPV-2025
SOBRE O PROJETO: nossa tarefa era editar o site de uma empresa nomeada de 'Social Connect', concertando erros do site e mudando informações incorretas sobre a empresa. Sobre essas informações incorretas: uma equipe anterior não entendeu o projeto da empresa e a descreveu de maneira equivocada, porém o texto já esteve correto, e teríamos que procurá-los num commit anterior.
NOSSA TAREFA: Um de nós fez fork no repositório original do professor CAIO OLIVEIRA da disciplina de WebDev. Depois, nos separamos em tarefas para concertar os erros e recuperar informações de outras branchs que haviam as informações corretas. Cada um fez uma branch com suas mudanças, contendo nelas, commits. 
Após isso, juntamos as branchs, então é possível vizualizar todas as alterações. 
FERRAMENTAS USADAS: em alguns momentos usamos IA para nos relembrarmos de alguns comandos (Copilot e ChatGPT)
